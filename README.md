# next-node-echarts
> Echarts for node

## install:
```bash
npm install -S afeiship/next-node-echarts --registry=https://registry.npm.taobao.org
```

## apis:
| api        | params | description        |
| ---------- | ------ | ------------------ |
| getDataURL | -      | get base64 string. |
| dispose    | -      | dispose resource   |

## usage:
```js
import NxNodeEcharts from 'next-node-echarts';

// code goes here:
```
